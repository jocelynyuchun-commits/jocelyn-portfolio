# checkvariables

Reference design to check: [UNTUCKit DLS](https://www.figma.com/design/oqfhkzHq6xa1STjHbS3Ofo/UNTUCKit_DLS_MCP?node-id=13777-5104&t=oBzym6QxNYNEscks-1)

Use the Figma MCP to get variable names and values for the selection. Then follow these steps:

1. **Determine what is being shared**
   - Is it a page, a component, or something else?

2. **Look at the UI or component as a whole**
   - Identify the page color
   - Identify all surfaces
   - Identify all text colors
   - Identify all borders
   - Identify all icon colors
   - Identify all secondary or tertiary components

3. **Review the variables in context**
   - Check surface variables against surface usage
   - Check border variables against border usage
   - Check text variables against text usage
   - Check icon variables against icon usage

4. **Call out what is wrong**
   - One line per issue
   - If nothing is wrong, state "nothing is wrong" or "all variables applied correctly"
